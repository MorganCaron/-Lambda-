What is ModularDom ?

ModularDom is a TypeScript framework designed to facilitate the creation of single-page web applications.
It allows you to create your own custom HTML elements and assign a state and behavior to them.

Why use it ?

It stands out from its competitors by its ease of use and its performance.
You can compare it to React or Angular, with the difference that ModularDom doesn't need to generate a DOM (Document Object Model). It is therefore much more powerful and its source code is very minimalist.
